# Sample Data for CARE-AI System
# ================================

This directory contains sample datasets for testing and demonstration purposes.

## Files:

### sensor_data.json
Sample accelerometer and gyroscope readings for fall detection testing.

### health_data.json
Sample health metrics including heart rate, blood pressure, and sleep data.

### user_data.json
Sample user profiles for system testing.

## Usage

These files can be used to:
1. Test the ML models without real sensor data
2. Demonstrate the system capabilities
3. Train and validate fall detection algorithms
4. Generate synthetic data for load testing
